这是一个基于微信公众号外卖订餐系统的客户端的前端实现

实现框架：vue

这个项目的后台代码可以在这个地址找到：https://github.com/philir/online-sell

代码运行：
window版本：把代码下载下来，在cmd命令行下，切换到到该项目的目录下，执行命令：npm run dev，就可以正常运行，前提是，已经安装好vue的相关依赖
